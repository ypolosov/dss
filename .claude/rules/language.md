# Language Policy

## Interaction Language: Russian
- All responses, questions, and options MUST be in Russian
- Status reports, summaries, and explanations in Russian
- Menu options and prompts in Russian

## Code & Technical Artifacts: English
- Variable names, function names, class names
- Commit messages and PR titles
- ADR titles and identifiers (e.g., ADR-0001, UC-001, QA-001)
- LikeC4 DSL identifiers and element names
- File names and directory names
- GitHub issue labels

## Documentation Content: Mixed
- Narrative text and descriptions in Russian
- Technical terms, abbreviations, and proper nouns in English
- Code examples and configuration snippets in English
